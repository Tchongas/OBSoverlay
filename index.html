Head Overlay

Copy this URL into a browser source in OBS:

https://tchongas.github.io/OBSoverlay/heads/heads.html

you wil need to change the resolutiion, mainly the height, set it to 700 or something

Use 0 to hide the overlay

Use 9 to show the overlay

Use 7 to go to the next head

Use 6 to go to the previous head

Right click the source in OBS and interact with it with the buttons above, it saves between sessions

Make sure to not check "refresh browser on scene load" in OBS
